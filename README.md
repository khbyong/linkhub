# linkhub

A tiny GitHub Pages link hub (QR-friendly).

## Deploy
1. Create a GitHub repo named `linkhub` (public recommended for Pages).
2. Push this folder contents to `main`.
3. In GitHub → Settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main` / `/root`
4. Your site will be at: `https://<your-username>.github.io/linkhub/`

## Update links
Edit `config.js`.
